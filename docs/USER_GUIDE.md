## 🚀 快速开始

### 第1步：下载 Launcher

打开管理后台网页，点击首页的 **「下载 Launcher」** 按钮，下载安装包。

### 第2步：启动 Launcher

1. 解压下载的 `OpenClawLauncher-win-x64-v1.0.2.zip`
2. 进入解压后的目录
3. 双击运行 `start.bat`
4. 等待终端显示 `Launcher 服务已启动，端口: 3003`

### 第3步：安装 OpenClaw

回到管理后台网页，点击 **「安装及配置」** → **「立即安装」**，系统自动完成安装。

### 第4步：选择配置模板

安装完成后，右侧选择一个共享配置模板快速配置，或开启 **Token 代理** 共享服务端模型（无需自己配置 API Key）。

### 第5步：配置飞书通道

点击 **「飞书消息通道」**，按引导完成飞书机器人配置，支持扫码快捷配置。

### 第6步：启动 Gateway

点击 **「日常运营」** → **「启动」**，启动 Gateway 服务。启动后即可通过飞书与 AI 助手对话。

### 第7步：输入邀请码

首次使用时，在飞书中向 AI 助手发送邀请码，激活 **1 万 Token** 体验额度，即可开始对话。

## 🔍 常见问题

### Q: Launcher 下载后怎么启动？

解压安装包，双击 `start.bat`，等待终端显示 `Launcher 服务已启动，端口: 3003` 即可。
要记得start.bat的位置，可以发送到桌面快捷方式

### Q: 网页显示「Launcher 未启动」？

确认已运行 `start.bat`，且终端显示端口 3003 已启动。启动后网页会自动检测到。

### Q: 邀请码怎么获取？

联系管理员获取邀请码。输入邀请码后可获得 1 万 Token 体验额度。

### Q: Token 用完了怎么办？

1 万 Token 体验额度用完后，前往 OpenClaw 配置自己的模型和 API Key，解锁完整功能。

### Q: 如何快速体验？

启动页面，启动gateway之后，点击打开管理控制台，可以直接在openclaw控制台的会话中进行对话。

### Q: 想用更专业的功能怎么办？

体验满意后，前往 OpenClaw 官网配置专属模型，解锁完整能力：
- 自定义模型选择
- 自定义 Agent 专长
- 自定义工作流程
- 更多高级配置
