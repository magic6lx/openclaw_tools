# OpenClaw Tools 开发环境启动命令

## 一键启动所有服务 (Windows)

```batch
d:\Tu工作同步\My工作同步\openclaw_tools\start.bat
```

启动后访问：
- Web界面: http://localhost:3001
- API: http://localhost:3002
- Launcher: http://localhost:3003

---

## 单独启动各服务

### 1. 启动后端 (端口3002)

```batch
cd d:\Tu工作同步\My工作同步\openclaw_tools\server
node src/index.js
```

### 2. 启动前端 (端口3001)

```batch
cd d:\Tu工作同步\My工作同步\openclaw_tools\client
npm run dev
```

### 3. 启动Launcher (端口3003)

Launcher 是 Node.js 运行时，自动管理本地 OpenClaw：

```batch
cd d:\Tu工作同步\My工作同步\openclaw_tools\launcher
npm run dev
```

---

## 停止所有服务

```batch
d:\Tu工作同步\My工作同步\openclaw_tools\stop.bat
```

---

## 构建 Launcher

仅当需要重新构建Launcher时才执行：

```batch
cd d:\Tu工作同步\My工作同步\openclaw_tools\launcher
npm install
npm run build
```

构建产物：`launcher\dist_new\OpenClawLauncher-v1.0.0-win-x64.zip`

## 项目结构

```
openclaw_tools/
├── client/          # 前端 React 应用 (端口3001)
├── server/           # 后端 Node.js API (端口3002)
├── launcher/         # Launcher 托盘应用 (端口3003)
├── config/           # 配置文件
├── docs/             # 文档
└── openclaw/        # OpenClaw 客户端
```

---

## 配置文件位置

- 后端: `server/src/index.js`
- 前端: `client/vite.config.js`
- Launcher: `launcher/src/index.js`
- 环境变量: `config/.env`
