# OpenClaw Tools 部署指令

## 系统架构

```
用户浏览器 → nginx:3001 → 前端静态文件
                        → /api → Express:3002 → MySQL
用户电脑   → Launcher:3003 (本地运行，不在服务器)
```

- **server/**: Express 后端，运行在 3002 端口
- **client/**: React 前端，构建后放入 server/public/，由 nginx 在 3001 端口托管
- **launcher/**: 本地客户端工具，分发给用户电脑运行，不部署在服务器
- **nginx**: 反向代理 + 前端静态文件服务

## 服务器部署

### 1. 服务器初始化

```bash
# 1. 克隆代码
cd /opt
git clone https://github.com/magic6lx/openclaw_tools.git
cd openclaw_tools

# 2. 安装后端依赖
cd server && npm install && cd ..

# 3. 安装前端依赖并构建
cd client && npm install && npm run build && cd ..

# 4. 初始化数据库
mysql -u root -p < server/scripts/init-db.sql

# 5. 配置环境变量
cp config/.env.example server/.env
# 编辑 server/.env 填入数据库信息和其他配置

# 6. 配置 nginx
sudo cp nginx.conf /etc/nginx/sites-available/openclaw
sudo ln -s /etc/nginx/sites-available/openclaw /etc/nginx/sites-enabled/
sudo nginx -t
sudo systemctl reload nginx

# 7. 创建日志目录
mkdir -p server/logs
```

### 2. 使用 PM2 启动后端

```bash
cd /opt/openclaw_tools/server
pm2 start ecosystem.config.js --env production
pm2 save
pm2 startup
```

### 3. 更新版本

```bash
cd /opt/openclaw_tools
git pull

cd server && npm install && cd ..
cd client && npm install && npm run build && cd ..

pm2 restart openclaw-server
```

## 环境变量配置

编辑 `server/.env`：

```bash
PORT=3002
NODE_ENV=production

DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=openclaw_tools
DB_USER=root
DB_PASSWORD=your_password

JWT_SECRET=your_secret_key_here
JWT_EXPIRES_IN=24h

LOG_LEVEL=info
LOG_FILE_PATH=./logs

FRONTEND_URL=http://YOUR_SERVER_IP:3001
```

---

## nginx 配置

编辑 `nginx.conf`，将 `YOUR_SERVER_IP` 替换为服务器 IP：

```bash
server {
    listen 3001;
    server_name YOUR_SERVER_IP;

    root /opt/openclaw_tool_server/server/public;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://127.0.0.1:3002;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

## PM2 管理

```bash
# 查看状态
pm2 status

# 重启服务
pm2 restart openclaw-server

# 查看日志
pm2 logs openclaw-server

# 停止服务
pm2 stop openclaw-server
```

## 常见问题

### 数据库连接失败
```bash
# 检查 MySQL 是否运行
sudo systemctl status mysql

# 测试数据库连接
mysql -u root -p -e "USE openclaw_tools; SHOW TABLES;"
```

### 前端页面空白
```bash
# 检查 nginx 是否正确配置
sudo nginx -t

# 检查静态文件是否存在
ls -la /opt/openclaw_tool_server/server/public/
```

### API 请求失败
```bash
# 检查后端是否正常运行
curl http://localhost:3002/api/config

# 检查 PM2 日志
pm2 logs openclaw-server --lines 50
```
