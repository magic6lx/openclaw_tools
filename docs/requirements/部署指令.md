# OpenClaw Tools 部署指令

## Git方式部署（推荐）

### 1. 服务器初始化

```bash
# 1. 克隆代码
cd /opt
git clone https://github.com/magic6lx/openclaw_tools.git
cd openclaw_tools

# 2. 安装依赖
npm install

# 3. 初始化数据库
mysql -u root -p < server/scripts/init-db.sql

# 4. 配置环境变量
cp config/.env.example config/.env
# 编辑 config/.env 填入数据库信息

# 5. 构建前端
cd client && npm install && npm run build && cd ..

# 6. 启动服务
npm start
```

### 2. 更新版本

```bash
cd /opt/openclaw_tools
git pull
cd client && npm install && npm run build && cd ..
pm2 restart openclaw-backend
```

---

## 环境变量配置

创建 `config/.env`：

```bash
PORT=3002
NODE_ENV=production
DB_HOST=127.0.0.1
DB_PORT=3306
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=openclaw_tools
JWT_SECRET=your_secret_key
```

## PM2 管理

```bash
# 查看状态
pm2 status

# 重启服务
pm2 restart openclaw-backend

# 查看日志
pm2 logs openclaw-backend
```
