# OpenClaw智能配置系统

## 项目简介

OpenClaw智能配置系统是一个帮助用户快速完成OpenClaw最佳配置的智能配置管理系统。

## 技术栈

- **前端**: React + Ant Design
- **后端**: Node.js + Express
- **数据库**: MySQL
- **ORM**: Sequelize

## 项目结构

```
openclaw_tools/
├── frontend/          # React前端应用
├── backend/           # Node.js后端服务
├── database/          # MySQL数据库脚本
├── docs/              # 项目文档
├── config/            # 配置文件
└── temp/              # 临时文件
```

## 快速开始

### 前端开发

```bash
cd frontend
npm install
npm start
```

### 后端开发

```bash
cd backend
npm install
npm run dev
```

## 功能特性

- 用户认证（基于邀请码）
- 配置向导
- 配置管理
- 日志管理
- 智能推荐
- 模版管理
